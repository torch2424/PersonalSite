• Chip Go

    ChipGo is a chip-8 computer emulator, that can be used to run many of the various programs or games written for the chip-8. In this project, I had to handle and read opcodes from chip-8 program files, and then translate the audio and video to openlGL to emulate the chip-8.

    ChipGo was built using Go, and open-source libraries such as Kingpin, and gosfml.
