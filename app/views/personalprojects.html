Personal Projects

These are a collections of some of my personal favorite projects that I have created/contributed to in my free time. I also have a lot more projects available open-sourced on my Github.

  • Flyxer

    Flyxer is a live performance music software that allows for loop switching "on the fly" with an easy to use interface. Along with a large collection of effects to edit your sound as it is playing.


    Flyxer was developed at Hacktech 2016 in 36 hours, and won the award, "Coolest Developer Tool". In my team of three, Julian Poyourow, Alexis Luter, and myself, I created the user interface, and user experience of the application, as well as handled the effects being applied to the JavaScript audio objects.


    Flyxer was built using Electron, NodeJS, and AngularJS. As well as the third-party JavaScript libraries, WAD (Which I have contributed to!), and Tuna.

  • Stats Monitor Widget

  • Super Punch Legacy

    Super Punch Legacy is a top-down cross-platform retro style arcade game, where a hero must punch to defeat waves of exploding enemies and deadly bosses.


    I developed sound and logic for the game, where visual art was created by members of Team No Comply Games. I handled all development of the game in Unity and implemented features such as, Camera Shake, Player Leveling, Player Save Management, Enemy Patterns, Character/Map Selection, and Game Physics.


    Beta builds are coming very soon!

  • linkDrops

    linkDrops is a website that will store all of those links you wanted to go back to, but never could, in an easy to use interface. linkDrops is perfect for things like youtube videos, articles, and repos!


    linkDrops' frontend is developed using JavaScript and CSS, and handls things such as keeping the link grid in order, sending requests to the backend to create accounts, save and manage user's links, and send requests to APIs provided by Noembed, Soundcloud, Spotify, and various other services to create iframes, and embeds of a user's link once they hover over the link. I had a bit of help from Julian Poyourow, as he mostly developed the back end, but we help one another in both the front and back end.


    linkDrops was built in the "MEAN" stack. Meaning it was built using powerful, modern day web development which includes: MongoDB, Express, AngularJS, and NodeJS. It was styled using Sass, a powerful css framework. Also, some other css libraries such as Animate.css and Angular Material. linkDrops also implements some third-party libraries such as Gridify, and Angular-inview.
