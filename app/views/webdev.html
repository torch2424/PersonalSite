Web Development

I have created/contributed to these websites and web apps as personal projects, or contracted work. I primarily use AngularJS and Sass for my frontend web development. I also develop using Ionic, and Electron to create multiplatform mobile applications. The projects I can share publicly, can be found on my Github.



  • linkDrops

    linkDrops is a website that will store all of those links you wanted to go back to, but never could, in an easy to use interface. linkDrops is perfect for things like youtube videos, articles, and repos!


    In linkDrops, I formulated the original app idea. I also developed the UI and UX of the application. I mostly developed the frontend using Javascript and CSS, and handled things such as keeping the link grid in order, sending requests to the backend to create accounts, save and manage user's links, and send requests to APIs provided by Noembed, Soundcloud, Spotify, and various other services to create iframes, and embeds of a user's link once they hover over the link. I had a bit of help from Julian Poyourow, as he mostly developed the backend, but we help one another in both the front and back end.


    linkDrops was built in the "MEAN" stack. Meaning it was built using powerful, modern day web development which includes: MongoDB, Express, AngularJS, and NodeJS. It was styled using Sass, a powerful css framework. Also, some other css libraries such as Animate.css and Angular Material. linkDrops also implements some third-party libraries such as Gridify, and Angular-inview.

  • CCRA E-Compendium

    The CCRA E-Compendium is an interactive online e-book reader for the California Court Reporters Association. The application implements requests for various search functions, subscription based debit card charges, and user validation. It was developed as a contract job by Kondeo. However, the app is in its final stages of deployment, and will be released for web, Android, and iOS. A similar application created by Kondeo (that I contributed to!) can be found here


    As a team-member of Kondeo, I created the user interface and controller logic to implement the mentioned features above. These features include, sending requests to search for words, validate users of three different types, send requests to receive and edit pages of the compendium, and inform users of the subscription depending on the current date. I also did all of the design and styling of the application.


    CCRA E-Compendium was built using Ionic, AngularJS, and Sass. As well as several third-party libraries, such as Stripe, Animate.css, and Moment.JS.

  • Flyxer

    Flyxer is a live performance music software that allows for loop switching "on the fly" with an easy to use interface. Along with a large collection of effects to edit your sound as it is playing.


    Flyxer was developed at Hacktech 2016 in 36 hours, and won the award, "Coolest Developer Tool". In my team of three, Julian Poyourow, Alexis Luter, and myself, I created the user interface, and user experience of the application, as well as handled the effects being applied to the Javascript audio objects.


    Flyxer was built using Electron, NodeJS, and AngularJS. As well as the third-party Javascript libraries, WAD (Which I have contributed to!), and Tuna.

  • Nudge

    Nudge is a social network that revolves around your own events, gatherings and parties. And those that are happening around you! Made with Kondeo Development (Still in pre-alpha phases)


    Nude was conceptualized by Julian Poyourow, and he built out most of both the front and back end of the application. I contributed to the project by doing front end controller logic, styling of the application, and created an events route to display individual event details. As well as solved some navigation bugs where the incorrect text would display after a user logged in. Nudge was actually one of the first AngularJS projects I contributed to, and helped me fall in love with the framework.


    Nudge was built with AngularJS and PHP giving a very fast and clean user experience on the frontend, with a powerful and reliable backend. Nudge was styled using Sass, a solid css framework.

  • Crunchkey

    CrunchKey is a simple text calculator that works with a press of a key, and gives your answer INSTANTLY. Built with both developers and users in mind, It strives to be a calculator that will save time editing, solving, and calculating any number crunching. Simply enter a statement and have you answer shown to you real time as it is entered.


    Crunchkey was the first web development project I created, and was actually one of the most useful. I used this application to learn basic web development, and developed it entirely myself. I created the calculation logic (by parsing the input expression into a stack) and design of the app.


    Crunchkey was developed using Html, Javascript, and Babel
