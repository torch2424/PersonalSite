Web Development

I have created multiple websites, web apps, and scripts for contracted work, and for fun in my free time. Where the ones I can share publicly, could be found on my Github.



  • linkDrops.

    "linkDrops." is a website that will store all of those links you wanted to go back to, but never could, in an easy to use interface. "linkDrops." is perfect for things like youtube videos, articles, and repos!

    "linkDrops." was built in the "MEAN" stack. Meaning is was built using powerful, modern day web development which includes: MongoDB, Express, AngularJS, and NodeJS. And it was styled using Sass, a powerful css framework. Also, some other css libraries such as Animate.css and Materialize.css

  • Nudge

    "Nudge" is a social network that revolves around your own events, gatherings and parties. And those that are happening around you! Made with Kondeo Development (Still in pre-alpha phases)

    "Nudge" was built with AngularJS, and PHP. Which gives a very fast and clean user experience, with a powerful and reliable backend. And it was styled using Sass, a solid css framework.

  • Team No Comply Games Development Blog

This web page was built to host their development blogs, so fans and the media could follow the game as it is developed!

    "Team No Comply Games" development blog was built with WordPress, a powerful, and easy to use blogging platform, that allows for blogging, e-commerce, and various web site functionality thanks to its robust plugin system.

  • Crunchkey

    CrunchKey is a simple text calculator that works with a press of a key, and gives your answer INSTANTLY. Built with both developers and users in mind, It strives to be a calculator that will save time editing, solving, and calculating any number crunching. Simply enter a statement and have you answer shown to you real time as it is entered.

    It was developed using Html, Javascript, J Query, and Extensionizer
