Android Development

I developed these Android applications independently as personal projects, using the Android Studio IDE. All of these projects, can be found on my Github, and My Developer Page from the Google Play Store.

  • Feather

    Feather is a lightweight and minimalistic music and video player for android. Feather allows user to explore the file structure to construct playlists of files. Feather was rated as "10 best Android video player apps" by Android Authority, which stated, “...A bare bones player without the bells and whistles which may be what a lot of people are looking for”.


    I independently developed Feather using multiple Android libraries, such as, Handlers for multi-threading, Media Player for managing and Android audio, and various Java File usage and tricks to browse user files, and allow extracting .zip fles.

  • Stats Monitor Widget

    Stats Monitor Widget is a minimal, conky-like, system monitor that sits on an android device's home screen, and displays per-core CPU usage, memory usage, network information, and other statistics of the user’s device.


    I independently developed Stats Monitor, which features various android libraries, such as, Broadcast Receviers for listening to system wide events, Shared Preferences for saving user configurations, and App Widget Providers which allows connecting Remote Views to the widget, and making it available to be created by an Android launcher.

  • Decicisons

    Decisions is an application that will allow you to build a question and fill it with available answers, and will then select a random answer for you. It also has quick settings such as "Yes or No", and "Coin Flip".


    I independently developed Decisions and exercised the use of serializeable Java objects to save user questions and answers to be retrieved later on. This also implements Android In-App-Payments.

  • Txtnote

    TxtNote is an application that will save notes as .txt files on the users device to be backed up, or stored for later use. This application is actually the first Android application I have developed, and still works great!


    I independently developed TxtNote and implemented basic Android Activities, Intents, application file directories, and XML Layouts.
